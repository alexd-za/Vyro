CLIP FACTORY  —  START HERE
============================

What this is:
  A toolkit that turns APPROVED Vyro campaign footage into short-form clips for
  TikTok / Reels / Shorts, run by two AI agents (Claude Code + Codex) that share
  one set of instructions, skills, and memory.

Three steps to running:
  1) Open a terminal in this folder.
  2) Install system tools (Fedora):
        sudo dnf install -y python3 python3-pip git
        sudo dnf install -y https://mirrors.rpmfusion.org/free/fedora/rpmfusion-free-release-$(rpm -E %fedora).noarch.rpm
        sudo dnf install -y ffmpeg yt-dlp
  3) Set it up and check it:
        chmod +x clip
        ./clip setup
        ./clip doctor      # everything required should read [ok]

Then read:  GETTING-STARTED.md   (full walkthrough: first campaign -> clips -> publish)
Menu:       ./clip               (or ./clip help for all commands)

The one rule that protects everything:
  Clip only approved source. Disclose AI where required. Never let unattended
  automation post for you. A live account that earns beats a banned one that doesn't.
